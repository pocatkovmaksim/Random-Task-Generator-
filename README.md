# Random-Task-Generator-

GUI-приложение «Random Task Generator» (Генератор случайных задач)  сохранением истории  в JSON 
# функции программы


# автор
Початков Максим

# запуск программы, библиотеки
необходимые библиотеки: tkinter, random, json
python RandomTaskGenerator.py

# Пошаговая инструкция:

1. Добавьте новую задачу (например, «Прочитать статью», «Сделать зарядку»).

2. Нажимая на кнопку «Сгенерировать задачу»,  выбирается случайная задача.

3. История задач выводится в виде списка.
# Проверяйте корректность ввода:

-  При добавлении новых задач — не пустая строка.
